# BIDS Validator

**Category:** Neuroimaging - Preprocessing

## Overview

BIDS Validator is a neuroimaging tool used for brain imaging analysis and processing.

- **Website:** https://github.com/bids-standard/bids-validator
- **Platform:** Node.js
- **Languages:** Node.js
